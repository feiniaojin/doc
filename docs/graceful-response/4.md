# 第三方组件适配

## Swagger

需要满足下面两点：

- Graceful Response版本必须大于等于3.2.1

- application.yaml配置以下属性：

```yaml
graceful-response:
  exclude-packages:
    - springfox.**
```
## spring-boot-starter-actuator

需要满足下面两点：

- Graceful Response版本必须大于等于3.2.1

- application.yaml配置以下属性：

```yaml
graceful-response:
  exclude-packages:
    - org.springframework.boot.actuate.*
```
